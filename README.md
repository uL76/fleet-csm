# Fleet CSM

Fleet CSM adalah aplikasi berbasis Laravel 12, React JS, Inertia, Tailwind CSS, dan Vite yang digunakan untuk pengembangan sistem Fleet / CSM secara modular.

Project ini dibuat sebagai dasar migrasi dari sistem PHP native menuju framework modern agar lebih mudah dikembangkan, dirawat, dan dilanjutkan oleh programmer lain.

---

## 1. Teknologi yang Digunakan

Project ini menggunakan stack berikut:

- Laravel 12
- React JS
- Inertia JS
- Tailwind CSS
- Vite
- MySQL / MariaDB
- Laragon
- Git & GitHub
- VS Code
